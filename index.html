Abschlussarbeiten

Wenn Sie eine Abschlussarbeit bei mir schreiben wollen, lesen Sie sich zuerst die Seite Hinweise zum Verfassen einer Abschlussarbeit durch und nehmen Sie dann per E-Mail Verbindung mit mir auf. Formulieren Sie dabei Ihr geplantes Thema und Ihren Zeitplan so genau wie möglich.

Git et al.

Im Rahmen meiner Praktika müssen Sie Abgaben in von mir zur Verfügung gestellten Git-Repositories abgeben. Lesen Sie sich dazu unbedingt die Seite Versionskontrolle mit Git über SSH durch. Für die Erstellung des notwendigen SSH-Keys beachten Sie die Seite OpenSSH-Keys generieren und nutzen.

Sonstige Dienste

Wenn Sie bei mir Zugriff auf ein Git-Repository haben, sollten Sie dieses auch über cgit betrachten können. Loggen Sie sich dort mit der Kennung der Fakultät, die Sie auch in den Laboren nutzen, ein. Falls Ihr Projekt bei jedem Push automatisch gebaut wird, finden Sie es im Jenkins.
